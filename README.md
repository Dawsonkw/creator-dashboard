# Creator Dashboard

A fullstack developer learning project built with React, TypeScript, Vite, and Node/Express.

## Purpose
Building a real SaaS-style streaming analytics dashboard while learning fullstack architecture, 
API design, and eventually DevOps thinking. The goal is to evolve this from mock data into a 
fully live Twitch-integrated dashboard with persistence and deployment.

## Roadmap
- [x] Frontend layout, components, and custom hooks
- [x] Backend API with Express and service layer
- [x] Environment variable setup
- [ ] Twitch OAuth + real data integration
- [ ] Database persistence (PostgreSQL + Prisma)
- [ ] Charts and analytics
- [ ] Deployment + DevOps layer

## Stack
- Frontend: React, TypeScript, Vite, TailwindCSS
- Backend: Node.js, Express, TypeScript
